# Todo API (Node.js + Express)

A RESTful API for managing todos with CRUD operations.

## Features
- Create todo
- Get all todos
- Update todo
- Toggle completed status
- Delete todo
- Input validation
- Proper status codes

## Tech Stack
- Node.js
- Express.js

## API Endpoints

GET /todos  
POST /todos  
PUT /todos/:id  
PATCH /todos/:id  
DELETE /todos/:id
